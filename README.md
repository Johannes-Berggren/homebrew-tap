# Homebrew Tap

Custom Homebrew formulae by [@johannesberggren](https://github.com/johannesberggren).

## Installation

```bash
brew tap johannesberggren/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [gitgoblin](https://github.com/johannesberggren/gitgoblin) | A beautiful terminal-based Git dashboard |

### GitGoblin

```bash
brew install johannesberggren/tap/gitgoblin
```
