# Homebrew Tap

Custom Homebrew formulae by [@Johannes-Berggren](https://github.com/Johannes-Berggren).

## Installation

```bash
brew tap Johannes-Berggren/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [gitgoblin](https://github.com/Johannes-Berggren/gitgoblin) | A beautiful terminal-based Git dashboard |

### GitGoblin

```bash
brew install Johannes-Berggren/tap/gitgoblin
```
